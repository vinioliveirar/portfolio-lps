# Gym Products Landing Page

Landing page baseada no template Sellsmart para produtos de academia.

## Estrutura de Arquivos

```
lp-gym/
├── index.html                    # Arquivo principal
├── assets/
│   ├── css/                      # Arquivos CSS (8 arquivos)
│   ├── js/                       # Arquivos JavaScript (17 arquivos)
│   └── images/                   # Imagens e ícones (63 arquivos)
└── README.md                     # Este arquivo
```

## Imagens Faltantes

As seguintes imagens podem estar faltando e precisam ser adicionadas manualmente:

- `hero-bg.jpg` - Imagem de fundo do hero section
- `testimonials-bg.jpg` - Imagem de fundo da seção de depoimentos
- `interactive-process-image-1.jpg` até `interactive-process-image-4.jpg` - Imagens do processo interativo

Essas imagens podem ser baixadas do site original ou substituídas por placeholders.

## Como Usar

1. Abra `index.html` no navegador
2. Ou acesse via servidor local: `http://127.0.0.1:5500/lps/lp-gym/index.html`

## Notas

- Todos os caminhos foram ajustados para a estrutura local
- Links externos foram removidos ou substituídos por `#`
